Architecture Highlights

Two-Pass Architecture

Maintains a strict separation of concerns. Pass 1 uses the LLM to read raw text and structure it logically into an intermediate JSON schema. Pass 2 acts as a routing engine, applying deterministic rules to deduplicate and place that JSON content into exact markdown files.

Skip & Force Logic

Before running expensive extractions, it computes SHA-256 hashes of all input files (Spec, Plan, Reviews). It compares these against ~/.claude/code-review/history/.../<slug>.json. If all hashes match, extraction is bypassed cleanly unless --force is invoked.

Batch Deduplication

When running --batch <dir>, the system iteratively extracts specs, preserving the ADR number counter. At the end of the batch run, it runs a cross-spec duplicate check on all generated ADRs, keeping the most detailed version and discarding overlaps without renumbering.
