Failure Modes & Recoveries

Failure Condition System Response & Recovery Prompt
No history directory
System State		 Agent intervenes: "No history found. Run /stark-review to generate data." Graceful exit without throwing unhandled exceptions.
No matching records
Filter Edge-case		 Agent advises: "No records match filters. Try broader criteria." Retains session context.
Corrupt JSON in history
Data Integrity		 Script emits warning on stderr, automatically skips the corrupted payload file, and continues processing remainder.
Script not found
Installation Error		 Agent aborts execution and directs user: "Run install.sh to set up stark-skills".
