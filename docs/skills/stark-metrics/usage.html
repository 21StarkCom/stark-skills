Improvement Flags (Meta-Observations)

🆕 First-Time Execution

If this is the first time metrics have been run, prints: "Tip: run /stark-metrics periodically to track trends."

📈 Prompt Improvements

If the report shows an improvement since the last prompt change (e.g., lower False Positive rate), explicitly flags it as a win.

🚨 Climbing Failures

If agent failure rates are trending upwards over time, flags the trend as an urgent issue requiring operator attention.

Common Failure Modes & Recovery

Failure Condition System Response / Recovery
No history directory Fails with Exit 1. Suggests: "No history found. Run /stark-review to generate data."
No matching records Returns empty state. Suggests: "No records match filters. Try broader criteria."
Corrupt JSON in history Script warns on stderr, skips the corrupt file gracefully, and continues processing the rest.
Script not found Alerts user to setup missing environment: "Run install.sh to set up stark-skills"
