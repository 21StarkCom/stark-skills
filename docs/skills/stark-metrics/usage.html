Quick Start

/stark-metrics --repo my-service # filter to one repo
/stark-metrics --skill stark-review # filter to one skill
/stark-metrics --since 2026-03-01 # only recent data
/stark-metrics --json # machine-readable output

What You Get

Agent Scorecards

Per-agent (Claude, Codex, Gemini) breakdown of review performance: findings submitted, acceptance rate, false positive rate, and average review duration.

Finding Quality

Distribution of findings by severity and type. Tracks signal-to-noise ratio and highlights which domains produce the most actionable feedback.

Duration Trends

How long reviews take over time. Spots regressions (reviews getting slower) and improvements after prompt changes or config tuning.

Prompt Improvement Impact

Before/after comparison when prompts were updated via /stark-review-improvement. Shows whether changes actually reduced false positives.

Recommendations

Actionable next steps: "run /stark-review-improvement for domain X", "tune config threshold Y", "investigate agent Z failure spike". Numbered for quick selection.

Failure Alerts

Flags climbing agent failure rates as urgent. Detects when an agent is silently breaking across runs so you can investigate before it impacts reviews.

Common Patterns

Weekly Check-in

Run /stark-metrics --since 2026-03-18 at the start of each week to spot trends from the past week's reviews.

After Prompt Tuning

Run /stark-metrics --skill stark-review after using /stark-review-improvement to verify false positive rates dropped.

Repo Deep-Dive

Run /stark-metrics --repo my-service --json to pipe structured data into dashboards or further analysis.
