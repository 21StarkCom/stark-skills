--state STATE option all Filter: open, closed, merged, or all

--limit N option 20 Max PRs shown in all-PRs mode

Common Workflows

Check on a specific PR

/stark-pr-status 42

Full dashboard for PR #42: review rounds, findings breakdown by severity and outcome, participant list, signal-vs-noise analysis, and chronological timeline.

Triage open PRs

/stark-pr-status --state open

Summary table of all open PRs. Quickly spot stale PRs with no reviews, or PRs ready to merge. Skill auto-suggests next actions for each.

Review quality audit

/stark-pr-status --all --state merged

See signal-to-noise ratios across all merged PRs. Identify which domains produce the most actionable findings vs. noise. Feed into /stark-review-improvement.

Cross-repo dashboard

/stark-pr-status --repo GetEvinced/web-sdk --state open

Check PR status in any repo without switching directories. Combine with --json for scripting or piping into other tools.

Machine-readable export

/stark-pr-status --json --limit 50

JSON output for integration with dashboards, CI pipelines, or custom analytics. Each PR includes full metrics payload.

Spot stale reviews

/stark-pr-status --state open

PRs open > 7 days with no reviews are flagged. The skill suggests running /stark-review to unblock them.

GitHub API

PR metadata, review comments, review requests, CI check status, merge events, participants. Always available if auth is configured.

Stark-Review History

Local history from previous /stark-review runs: findings by domain, severity classifications, signal-vs-noise outcomes. Enriches the dashboard when available.
