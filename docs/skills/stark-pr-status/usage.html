Quick Start

Single PR Dashboard

/stark-pr-status 42

Full lifecycle view of PR #42: reviews, findings, participants, timeline, and signal analysis.

All PRs Summary

/stark-pr-status --all

Summary table of all PRs in the current repo. Add --state open to filter.

Cross-Repo Query

/stark-pr-status --repo GetEvinced/platform --state merged --limit 10

Query a specific repo regardless of your current directory.

Arguments & Flags
Argument Type Default Description
PR_NUMBER positional — Show full dashboard for a single PR
--all flag implied Show summary of all PRs (default when no number)
--repo ORG/NAME option auto-detect Override repository (e.g., GetEvinced/platform)
--state STATE option all Filter: open closed merged all
--limit N option 20 Maximum PRs to show in all-PRs mode
--json flag off Machine-readable JSON output

Common Patterns

Check Stale PRs

/stark-pr-status --state open

Find PRs that are open and may need attention. The skill flags PRs open >7 days with no reviews and suggests running /stark-review.

Review Quality Check

/stark-pr-status 15

Inspect signal-vs-noise ratio on a specific PR. High noise? The skill suggests /stark-review-improvement to tune prompts.

JSON for Scripting

/stark-pr-status --all --state merged --json

Machine-readable output for piping into dashboards, scripts, or further analysis.

Ready to Merge?

/stark-pr-status 28

If the PR is approved with green CI, the skill suggests /stark-pr-flow to ship it.

Error Handling

Exit 1 — No Repo Detected

Not in a git repo. Run from a repo directory or use --repo org/name to specify explicitly.

Exit 2 — PR Not Found

The PR number doesn't exist in the detected repo. Double-check the number or use --repo if you're targeting a different repo.

Exit 3 — GitHub API Error

Auth failure or rate limiting. Run install.sh to verify credentials, or wait a few minutes if rate-limited.

No Review History

GitHub data is shown but stark-review history is missing. Not an error — the PR may not have been reviewed with /stark-review yet.
