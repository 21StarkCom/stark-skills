Internal Logic & Heuristics

5 Analysis Dimensions

The skill evaluates documentation quality against these vectors:

  • Structure & Clarity: Scannability, header logic, redundancy.
  • Completeness: Tech stack, architecture, workflows, don'ts.
  • Accuracy & Freshness: Valid paths, existing commands, current versions.
  • Effectiveness for AI: Actionable instructions vs descriptive text.
  • Hierarchy Optimization: Right-sizing info to home, org, or project levels.

Output Data Structure

The generated report maps findings strictly by file path and includes:

  • 1-5 Score per dimension
  • Line-referenced issues
  • Drafted additions/removals
  • Hierarchy refactoring suggestions

Skill-Specific Observability

Adheres to standard protocols plus explicit metrics:

  • Count of files discovered/analyzed
  • Matrix of per-file dimension scores
  • Suggested mutation counts (adds/removes/moves)
  • Actual applied mutation count
