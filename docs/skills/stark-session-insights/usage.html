What Gets Measured

Session Volume

Prompt count, date range, active days, prompts per day, session count, and average session length. Missing sessionId values fall back to a 30-minute gap heuristic.

Command Usage

Extracts slash commands such as /stark-review, /commit, or /help, then ranks them by frequency.

Action Vocabulary

Counts action words like review, fix, test, push, deploy, refactor, debug, configure, and compare to show how work is framed.

Work Rhythm

Builds hourly activity bars, day-of-week breakdowns, and weekly trends so you can see when the project ramps up, peaks, sustains, or tapers.

Corrections & Friction

Captures prompts that look like course corrections: no, don't, actually, undo, and emphatic prompts ending with !.

Narrative Synthesis

Turns all metrics into a 3-5 paragraph interpretation of project type, working style, failure modes, evolution, and concrete recommendations.

Decision Points & Edge Cases

No History File

If ~/.claude/history.jsonl is missing or empty, the skill aborts with a direct error instead of silently succeeding.

No Matching Project

If --project matches nothing, the skill returns available short names so the user can rerun with a valid partial path.

Malformed JSONL

Broken lines are skipped and counted as warnings. Valid history continues to be processed.

Freshness Checks

Existing reports are skipped when the metadata entry count matches the current project entry count, unless --refresh is supplied.

Minimal Data

Projects with only one prompt still get reports, but average session duration is shown as zero and optional sections may be omitted.

Privacy Scope

The skill reads local history and writes local markdown only. It does not modify history, send data elsewhere, or compare across users.
