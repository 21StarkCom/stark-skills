stark-session-insights

Analyze Claude Code session history to extract usage patterns, skill invocations, action frequencies, corrections, and preferences — grouped by project.
COMPLEXITY: MEDIUM

How to Invoke

Command

/stark-session-insights [--project <name>] [--refresh]

Natural Language: "Give me session insights", "analyze my sessions", "usage patterns", or "what do I do most?"

--project <name>

Filter to a specific project (case-insensitive partial match on the path). If omitted, processes all projects found in your history.

--refresh

Force regeneration of the insights even if the file exists and the number of entries hasn't changed.

Execution Flow

Reads user's local history data.
~/.claude/history.jsonl
Raw chat interactions and timestamps
Parse & Group
Group history by project & slugify paths
Filters by --project if provided.
Skips unchanged projects unless --refresh is used.
Calculates 15+ different metrics,
trends, and patterns via embedded Python.
Deep Data Analysis
Categorize intents, find corrections, track rhythms
Generate Insights Directory
~/.claude/code-review/insights/sessions/
Creates Markdown reports ready for the user to read.

What Gets Analyzed (The Insights)

⏱️ Activity & Rhythms

  • Day of Week: Are you a weekend warrior or weekday focused?
  • Hourly Spread: Your peak hours for using Claude Code.
  • Weekly Trends: Project lifecycle (ramp-up, peak, tapering).

🛠️ Skill & Action Usage

  • Top Commands: Your most used /stark-* commands.
  • Action Words: Frequency of "refactor", "debug", "test", etc.
  • Skill Evolution: How your workflow shifts over weeks.

⚠️ Corrections & Friction

  • Pain Points: Detects when you say "no", "stop", "wrong", or "actually".
  • Categorization: Groups errors into Logic, UX, Process, or Frustration.
  • Recommendations: Auto-suggests fixes for recurring workflow friction.

🧠 AI Narrative Synthesis

  • Session Shape: Are sessions mostly "Build/Ship" or "Debug"?
  • Key Requirements: Extracts your longest, most detailed prompts.
  • Prose Analysis: Writes a customized 3-5 paragraph "so what?" narrative.

Generated Outputs

All outputs are generated locally in ~/.claude/code-review/insights/sessions/. No data is sent externally.

Artifact Description Key Sections Included
{project-slug}.md Deep-dive analysis for a single project (e.g., evinced--infra-pulse.md). Activity Profile, Skill Usage, Corrections Log, Narrative Synthesis, Actionable Recommendations.
