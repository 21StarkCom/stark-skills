Internal Architecture

Version Source of Truth

Git tags (v{M}.{m}.{p}) are canonical. __init__.py is the runtime mirror — bumped every release. pyproject.toml is static, never touched.

git tags → canonical __init__.py → runtime pyproject.toml → static

Bump Auto-Selection Logic

When no explicit argument given, inspects CHANGELOG categories:

  • patch — only ### Fixed entries
  • minor — any ### Added entries
  • major — ### Changed with breaking changes

Explicit $ARGUMENTS always overrides.

Auth Model

unset GH_TOKEN at entry ensures gh falls back to native PAT auth. Tags, commits, and GitHub Releases all show as the user, not a bot. Inverse of /stark-review which uses bot tokens.

CHANGELOG Contract

Requires ## [Unreleased] with content. On release, entries move to ## [vX.Y.Z] - DATE. The [Unreleased] header remains but is emptied. Missing or empty = hard abort.

Failure Modes & Recovery
Failure Detection Point Recovery
Not on main Step 1 — Pre-flight Prompt: git checkout main
Dirty working tree Step 1 — Pre-flight Prompt: stash or commit changes
No CHANGELOG.md Step 3 — Review Hard abort — nothing to release
Empty [Unreleased] Step 3 — Review Hard abort — nothing to release
Tag already exists Step 7 — Tag Error + suggest next version
Push conflict Step 8 — Push git pull --rebase origin main, retry
gh auth fails Step 9 — Release Verify gh auth status
Extension Points

Dynamic Versioning

Currently pyproject.toml is static. To switch: add setuptools-scm — reads tags at build time, eliminating manual __init__.py bumps.

New CHANGELOG Categories

Auto-bump handles Fixed, Added, Changed. New categories (Deprecated, Removed, Security) need mapping in Step 4's decision tree.

Observability Protocol

Follows standards/observability.md. Emits: version diff, entry counts by category, tag/release flags, push duration.

Invariants (Mistakes to Avoid)

  • Never bump pyproject.toml
  • Always bump __init__.py
  • Never set GH_TOKEN during release
  • Never release with empty [Unreleased]
  • Commit before tagging (not reverse)
