Component Architecture

Orchestrator Skill

The primary stark-review skill sequence that handles git isolation, state management, round logic, and post-processing.

Observability: Forces structured Task creation, timestamped log lines, and 5-min checkpoints directly to Claude Code output.

multi_review.py

Python dispatcher handling parallel execution. Dispatches 3 agents via subprocess. Sub-agents run isolated prompts on local diffs.

Flags: claude -p -, codex exec review --json, gemini -p <prompt> -o json

Classification Logic

Categorizes raw JSON findings dynamically against the checked-out branch code.

  • fix Target severity >= threshold AND exists
  • recurring Survives multiple rounds
  • false_positive Code pattern doesn't exist
  • noise Subjetive/Style/Contradicted
