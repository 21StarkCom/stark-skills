stark-update-deps

Dependency Version Updater
Usage Guide
Overview

What it does

Systematically audits every versioned dependency in a project, looks up current stable releases on official sources, identifies breaking changes, updates versions in-place, and rigorously verifies the result against web sources to prevent hallucinations.

How to trigger

Say: "update dependencies", "check for outdated packages", "upgrade versions", or "are my deps current?"

Key Value

Verification Phase: LLMs often hallucinate versions confidently. This skill verifies every single updated tag against live registries (PyPI, npm, Docker Hub) before finalizing the update.
