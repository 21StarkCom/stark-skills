Overview

This skill generates a comprehensive analytics report detailing which skills are invoked, how often, by which projects, and with what quality. It aggregates data from global history files and local project repositories to compute adoption curves, usage rankings, error rates, and actionable recommendations.

Report Content Capabilities

Usage & Adoption Rankings

Tracks invocation counts, unique projects, and 30-day trends (rising, stable, declining, new). Plots chronological adoption timelines to see when skills were first introduced to user workflows.

Quality Signals

Surfaces skills with high timeout rates (>20%), high failure rates (>10%), or those that frequently return empty results. Breaks down success rates by LLM agent (Claude, Codex, Gemini) where applicable.

Workflow Sequences

Identifies consecutive skill pairings within single sessions (e.g., /stark-review followed by /stark-review-improvement) to uncover emergent user workflows that could be automated into compound skills.

Actionable Recommendations

Automatically suggests actions: Remove dead skills, Promote underused high-quality skills, Fix buggy tools, and Combine common sequences.
