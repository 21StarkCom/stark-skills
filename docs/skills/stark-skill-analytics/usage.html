Arguments & Outputs

--skill <name>

Generates a deep-dive report specifically for the provided skill name (e.g., stark-review). Analyzes detailed usage trends, parameter variations, success/failure rate overtime for that single skill.

--format table | full

Controls output verbosity. table skips per-skill breakdowns and prints summary usage rankings. full (default) generates the complete adoption, quality signals, and sequence report.

Output Highlights

Recommendations: Actionable steps to remove dead skills, promote healthy ones, fix high-failure-rate skills, or combine common sequences.

Metrics Tracked

Usage & Adoption

Tracks how many times a skill is run, the variety of projects using it, 30-day trends (rising, declining, new, stable), and the ratio of parameterized vs. "quick" (argument-less) invocations.

Quality & Performance

Aggregates agent timeout rates, general error rates, empty result percentages, and total duration metrics to score the health of a given skill implementation.

Ecosystem Health

Checks the gap between configured skills (CLAUDE.md) and actual usage (history.jsonl) to flag Unregistered but used and Registered but dead skills.
